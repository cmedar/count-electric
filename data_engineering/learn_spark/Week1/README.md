# Week 1 — Apache Spark Fundamentals

Core concepts for the Databricks Certified Data Engineer Associate exam.

**Days 1–7 cover:**
- Spark architecture: driver, cluster manager, executors
- Jobs, stages, tasks, and the DAG execution model
- Narrow vs wide transformations and shuffle
- DataFrame API: transformations, aggregations, joins
- Spark SQL: temp views, window functions, Catalyst optimizer
- Reading and writing data: CSV, Parquet, Delta Lake basics
- Deployment modes: client vs cluster
