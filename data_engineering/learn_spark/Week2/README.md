# Week 2 — Advanced Transformations & Analytics

Core concepts for the Databricks Certified Data Engineer Associate exam.

| Day | Notebook | Topics |
|-----|----------|--------|
| 8 | `day_8.ipynb` | Aggregations — `groupBy`, `agg`, multi-function calls, `pivot`, HAVING equivalent, `describe()` |
| 9 | `day_9.ipynb` | All 7 join types (inner / left / right / full / cross / semi / anti), broadcast joins, column conflict resolution |
| 10 | `day_10.ipynb` | String functions, date & timestamp parsing/formatting/arithmetic, `regexp_replace`, `regexp_extract` |
| 11 | `day_11.ipynb` | Math functions, array operations (`explode`, `collect_list`, `array_contains`), maps, structs |
| 12 | `day_12.ipynb` | Row-at-a-time UDFs, Pandas (vectorized) UDFs, SQL registration, when not to use UDFs |
| 13 | `day_13.ipynb` | Window spec (`partitionBy`, `orderBy`, `rowsBetween`), ranking (`rank`, `dense_rank`, `row_number`), navigation (`lag`, `lead`), running aggregations |
| 14 | `day_14.ipynb` | `cache()` vs `persist()`, storage levels (`MEMORY_ONLY`, `MEMORY_AND_DISK`, `DISK_ONLY`), `unpersist()`, when caching helps vs hurts |
