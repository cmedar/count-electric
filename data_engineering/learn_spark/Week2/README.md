# Week 2 — Advanced Transformations & Delta Lake

Core concepts for the Databricks Certified Data Engineer Associate exam.

**Days 9–14 cover:**
- Aggregations: `groupBy`, `agg`, `pivot`, `rollup`, `cube`
- Window functions: `rank`, `dense_rank`, `row_number`, `lead`, `lag`, running totals
- Advanced joins: broadcast, semi, anti, cross joins
- UDFs and pandas UDFs (vectorized)
- Caching and persistence levels
- Delta Lake: time travel, `OPTIMIZE`, `VACUUM`, `MERGE INTO`, table history
